昨天在 Github 上看到了 [js-org/dns](https://github.com/js-org/dns/) 这个项目，之前正好考虑到明年域名到期就不再为这个过气的库单独花费一个域名了，所以说干就干趁机就来了一发 pull request 把项目的域名改成了 [elf.js.org](http://elf.js.org/)，看起来也非常合适。这东西反正也没什么人用，挂在里面就当个纪念吧。
